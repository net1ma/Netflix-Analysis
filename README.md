# Netflix Catalog Analysis

## Описание проекта

В рамках проекта проведён комплексный анализ каталога Netflix: от сбора и объединения данных из разных источников до построения рекомендательной системы для поиска похожих фильмов и сериалов. Использованы современные методы обработки данных, визуализации и машинного обучения.

## Структура проекта

| **Файл** | **Описание** |
|-----------------|-----------------------------------------------------------------|
| `datasets_merging.ipynb` | Сбор и объединение данных из нескольких источников |
| `data_transformation_and_cleaning.ipynb` | Очистка, преобразование и подготовка данных к анализу |
| `analysis.ipynb` | Исследовательский анализ, визуализация, выявление закономерностей |
| `similar.ipynb` | Построение датасета с эмбеддингами, поиск похожих фильмов/сериалов |

## Ключевые задачи

- Сбор данных: интеграция информации из разных источников.
- Очистка и трансформация: обработка пропусков, дубликатов, приведение форматов.
- Исследовательский анализ: визуализация распределения жанров, рейтингов, популярности и других характеристик.
- Поиск похожих: построение эмбеддингов, реализация поиска по контенту и признакам.
